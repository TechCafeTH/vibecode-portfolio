1. ไอเดียดิบ + ปัญหา + กลุ่มผู้ใช้
เว็บโปรไฟล์ส่วนตัว (personal profile / portfolio site) สร้างจากข้อมูลใน LinkedIn เพื่อใช้สมัครงาน
ปัญหาที่แก้: ตอนนี้ส่งเรซูเม่เป็นไฟล์ PDF ซึ่งดูตายตัว แก้ทีต้องส่งใหม่ทุกครั้ง และไม่ได้สะท้อนตัวตนเท่าไหร่ อยากมีลิงก์เดียวที่ส่งให้ HR หรือแปะใน LinkedIn/อีเมลได้เลย เปิดแล้วเห็นโปรไฟล์ ผลงาน และช่องทางติดต่อครบ
กลุ่มผู้ใช้: คนหางาน/เปลี่ยนงานสายดีไซน์ (เจ้าของเว็บ) ฝั่งคนดูคือ HR และ hiring manager ที่เปิดดูตอนพิจารณาใบสมัคร

2. ฟีเจอร์หลัก (MVP)
Hero section: ชื่อ ตำแหน่ง ประโยคแนะนำตัวสั้นๆ รูป และปุ่มติดต่อ
About: ย่อหน้าแนะนำตัวเอง
Experience: ไทม์ไลน์ประสบการณ์ทำงาน (ตำแหน่ง บริษัท ช่วงเวลา สิ่งที่ทำ)
Skills: ลิสต์ทักษะแบบ tag
Portfolio/Projects: การ์ดผลงาน (ชื่อ คำอธิบายสั้น ลิงก์)
Contact: อีเมล + ลิงก์ LinkedIn/social
ปุ่ม "ดาวน์โหลด CV (PDF)" สำหรับ HR ที่อยากได้ไฟล์

3. ข้อมูลที่ต้องแสดง
ข้อมูลโปรไฟล์ (ชื่อ ตำแหน่ง bio รูป อีเมล โซเชียล), รายการประสบการณ์ทำงาน, รายการทักษะ, รายการผลงาน ทั้งหมดดึงโครงมาจาก LinkedIn PDF เป็นเว็บ อ่านอย่างเดียว ไม่มีระบบให้คนนอกกรอกอะไร

4. สไตล์
Clean & Minimal โทนสว่าง อ่านง่าย เป็นมืออาชีพ เน้นตัวอักษรชัด มี whitespace เยอะ เพราะคนดูคือ HR ที่อยากสแกนเร็วๆ (ถ้าเป็นสายครีเอทีฟจะเติม accent สีได้นิดหน่อย)
